# modern-cljs

An implementation of [Modern ClojureScript](https://github.com/magomimmo/modern-cljs)


## Usage

```
# Build Project
$ lein do cljsbuild clean, cljsbuild once

# Run bREPL
$ lein trampoline cljsbuild repl-listen

```
